# Django_tutorial

## Install Homebrew 
<code>
/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
</code>

## Install Python3
<code>
brew install python3
</code>

## Install pipenv
<code>
pip3 install pipenv
</code>

## Install Django(ver.2.1)
<code>
cd target_DIR
pipenv install django==2.1
</code>

## Run pipenv
<code>
pipenv shell
</code>

## Start project
<code>
django-admin startproject project_name .
</code>
